# Astralcast 2.0 Vision

Astralcast is a celestial tarot experience.

Every tarot card exists as a star within a living night sky.

The user enters birth information and a question.

The user enters the Starfield.

By touching and holding the screen, stars gather and orbit around the user's finger.

The user focuses on their intention.

Upon release, the stars scatter back into the cosmos.

The user selects stars.

The selected stars reveal archetypes.

The reading is interpreted through an AI Oracle.

Future features:

- Voice Oracle
- Astrology Integration
- Personal Cosmos
- Reading History
- AI Follow-Up Conversations
